# Imam Games

Aplikasi web permainan edukasi anak-anak. Termasuk beberapa mini-game interaktif:

- Tabak Gambar
- Tebak Nama
- Tebak Negara
- Quiz Kata

Fitur utama:

- Timer 5 menit
- Skor interaktif
- Tombol mulai ulang dan ganti game
- Antarmuka sederhana dan ramah anak

Cara menjalankan:

1. Buka `index.html` di browser.
2. Pilih permainan.
3. Tekan tombol **Mulai Permainan**.
4. Jawab soal sebelum waktu habis.

Pengembangan lebih lanjut:

- Menambahkan gambar nyata
- Menyimpan skor tertinggi
- Menambahkan lebih banyak kategori permainan
